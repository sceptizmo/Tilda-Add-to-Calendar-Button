# Tilda Add to Calendar Button

Скрипт для платформы [Tilda](https://tilda.cc), добавляющий кнопкам функционал «Добавить в календарь». При клике открывается меню с выбором календаря: Google Calendar, Apple / iCal, Outlook 365, Outlook Live.

## Возможности

- **Google Calendar** — открывает страницу создания события
- **Apple / iCal** — скачивает файл `.ics` для добавления в календарь
- **Outlook 365** — открывает форму создания события
- **Outlook Live** — открывает форму создания события

Работает со стандартными блоками Тильды с кнопкой. Протестировано на блоке **BF101 Кнопка** — это блок с одной кнопкой, на нём скрипт работает корректно.

## Установка

### Шаг 1. Назначьте класс блоку с кнопкой

В редакторе Тильды выберите блок с кнопкой и добавьте класс **`uc-cal-but`** в настройках блока (раздел «Дополнительно» → «Класс CSS»). Класс назначается всему блоку.

### Шаг 2. Добавьте HTML-блок

1. Добавьте на страницу блок **«HTML-код»**
2. Скопируйте содержимое файла `embed.html` и вставьте в блок
3. Отредактируйте данные события в блоке **НАСТРОЙКИ СОБЫТИЙ** (в начале скрипта):
   - `EVENT_TITLE` — название мероприятия
   - `LOCATION` — место проведения (например: «Zoom», «Онлайн», адрес)
   - `START_ISO` — дата и время начала в формате ISO 8601 (например: `2026-01-15T19:00:00+03:00`)
   - `END_ISO` — дата и время окончания
   - `DESCRIPTION` — описание мероприятия (можно многострочное)

### Формат даты

Используйте формат ISO 8601: `ГГГГ-ММ-ДДTЧЧ:ММ:СС+ЧЧ:ММ`

Примеры:
- `2026-01-15T19:00:00+03:00` — 15 января 2026, 19:00 по Москве
- `2026-02-13T17:00:00+01:00` — 13 февраля 2026, 17:00 по Берлину

## Особенности

- **Мобильные устройства**: во встроенных браузерах (например, в приложениях соцсетей) скачивание ICS может не работать. Скрипт показывает подсказку: открыть страницу в Safari, Chrome или на компьютере.
- **Несколько блоков**: можно назначить класс `uc-cal-but` нескольким блокам — все будут работать с одними и теми же данными мероприятия.
- **Блок BF101 Кнопка**: скрипт проверен на блоке BF101 Кнопка (блок с одной кнопкой). В других блоках с кнопками может работать аналогично, но структура DOM у блоков Тильды различается — при проблемах проверьте наличие элементов `.t-container` и `.t-btnflex` в нужном блоке.
- **Стили выпадающего меню**: по умолчанию меню выпадает снизу под кнопкой. Под конкретный стиль сайта или лендинга может потребоваться изменить оформление (цвета, шрифты, рамки) или позицию — например, чтобы меню появлялось справа. Стили задаются в блоке «Меню календаря» в коде; при необходимости можно доработать их самостоятельно.
